msiempy API Documentation

Module Index

  • msiempy - The pythonic way to deal with the McAfee SIEM API
    • msiempy.__utils__ - Keep this module and imports for backwards compatibility. Please use msiempy.core.utils
    • msiempy.__version__ - Project version and meta informations.
    • msiempy.alarm - Provide alarm management. Define AlarmManager and Alarm.
    • msiempy.core - The core objects of the library: NitroSession, NitroConfig, NitroError and other.
      • msiempy.core.config - Provide simple configuration object with a few custom properties. Define NitroConfig.
      • msiempy.core.query - Base query class with timerange handling. Define FilteredQueryList.
      • msiempy.core.session - HTTP level interface to the ESM API. Define NitroSession and NitroError.
      • msiempy.core.types - Base classes for SIEM interface objects and generic definitions. Define NitroObject, NitroDict and NitroList.
      • msiempy.core.utils - A few quick static util methods.
    • msiempy.device - Provide ESM, Receiver and Datasource management. Define ESM, DevTree and DataSource.
    • msiempy.event - Provide event management. Define EventManager, Event, FieldFilter, GroupFilter, GroupedEventManager, GroupedEvent.
    • msiempy.watchlist - Provide watchlist management. Define WatchlistManager and Watchlist.

def status(self): (source)

Returns: `dict` ESM statuses including : - `cpu`, example: `Avail: 7977MB, Used: 7857MB, Free: 119MB` - `hdd`, example: `sda3 Size: 491GB, Used: 55GB(12%), Available: 413GB, Mount: /` - `ram` - `callHomeIp` - `autoBackupEnabled` - `autoBackupHour` - `autoBackupDay` - `backupNextTime` - `backupLastTime` - `rulesAndSoftwareCheckEnabled` - `rulesAndSoftNextCheck` - `rulesAndSoftLastCheck` Other functions exist to return subsets of this data also. .. note:: Uses internal API method `SYS_GETSYSINFO`

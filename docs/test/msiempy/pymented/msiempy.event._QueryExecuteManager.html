Part of msiempy.event View Source (View In Hierarchy)

Known subclasses: msiempy.event.EventManager, msiempy.event.GroupedEventManager

Abstract class to handle common filters properties that grouped and non-grouped queries share. As well as providing helper functions to wait the query and get the results (see source code). Only Events query are currently supported.
